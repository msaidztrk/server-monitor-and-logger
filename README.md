# Server Monitor & Logger

Uzak sunuculardan Python Agent ile veri toplayıp, Laravel API üzerinden React (Vite) dashboard'una anlık yansıtan Full-Stack izleme sistemi.

## Tech Stack
- **Backend:** Laravel 11, MySQL, Redis, Sanctum, Reverb
- **Frontend:** React (Vite), TypeScript, Tailwind CSS, Recharts
- **Agent:** Python 3 (`psutil`, `requests`)
